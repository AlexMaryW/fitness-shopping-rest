Fitness Shopping REST API’s Documentation#

Fitness Shopping REST API is an open-source backend designed to support both Fitness App and Shopping Website. It provides essential features for user management, fitness tracking, and shopping-related functionality.

The User module allows you to register users, log them in, and authenticate HTTP requests using tokens, and securely store their information.

The Fitness module allows you to track physical activities, meals, and goals related to calories (kcal) and macronutrients.

The Shopping module allows you to to add and categorise, write product reviews, and managing messages.

Important

API Base URL: https://example.com/api/

All requests should be made to this base URL, followed by the relevant endpoint. For example: GET https://example.com/api/activities/

Warning

Token-based authentication via BasicAuth is required for most HTTP endpoints. Tokens are provided upon successful login via the User module.

Status Codes

This API follows standard HTTP status codes.
