Food Archetypes#

This module allows you to create, view, search for, modify, and delete food archetypes.

Warning

All endpoints in this module require authentification token provided in the Authorization header.

Food Archetype Information#

Each food archetype can be assigned the following information:

Attribute

Type

Required

Description

Name

str

No

The food archetype’s name, e.g. “carrot”.

Measure

str

No

The food archetype’s measure, e.g. grams or cup.

Calories

float

No

The amount of calories (kcal) in a food archetype.

Protein

float

No

The amount of protein in a food archetype, measured in grams.

Carbohydrates

float

No

The amount of carbohydrates in a food archetype, measured in grams.

Fat

float

No

The amount of fat in a food archetype, measured in grams.

user_id

int

Yes

The id of the currently logged in user is automatically assigned to the food archetype.

Note

The default value of each attribute, except user_id, is None.

HTTP Endpoints#

GET /foods/#

Allows you to view all food archetypes.

Status Codes

If the food archetype hasn’t been found, 404 status code is returned.

Example request:

GET /foods/
Host: example.com
Accept: application/json
x-access-token: eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VybmFtZSI6ImphbmVkb2U0NCIsImV4cCI6MTc0NDYyOTM3OX0.43dgIGUbKHtbDs5XqzCl_MFsKFgMyFo_SyKY1sLmCu4

Allows you to search for a food archetype. If successful, the endpoint returns the food archetype’s information.

Type

Required

Description

Status Codes

POST /foods/#

Allows you to add a new food archetype. For each unique food archetype, you can add the following information.

Attention

This endpoint requires admin status.

Status Codes

This endpoint can return 403 or 400 status code.

Example request:

POST /foods/
Host: example.com
Accept: application/json
x-access-token: eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VybmFtZSI6ImphbmVkb2U0NCIsImV4cCI6MTc0NDYyOTM3OX0.43dgIGUbKHtbDs5XqzCl_MFsKFgMyFo_SyKY1sLmCu4

  {
"name": "greek yoghurt",
"measure": "cup",
"calories": 146,
"carbohydrates": 9.8,
"fat": 4.9,
"protein": 20.0
  }

Example response:

  {
"id": 20,
"message": "Food created!"
  }

PUT /foods/<food_id>#

Allows you to modify a food archetype’s information (except the user_id).

Parameters#

Attribute

Type

Required

Description

food_id

int

Yes

The id the food archetype you’re looking for.

Attention

This endpoint requires admin status.

Attention

The request is successful only when the food_id in the URL is assigned to your user_id.

Status Codes

This endpoint can return 403, 404, or 400 status code.

Example request:

PUT /foods/20
Host: example.com
Accept: application/json
x-access-token: eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VybmFtZSI6ImphbmVkb2U0NCIsImV4cCI6MTc0NDYyOTM3OX0.43dgIGUbKHtbDs5XqzCl_MFsKFgMyFo_SyKY1sLmCu4

  {
"name": "greek yoghurt",
"measure": "100g",
"calories": 59,
"carbohydrates": 3.6,
"fat": 2.0,
"protein": 10.0
  }

Example response:

  {
"message": "Food updated!"
  }
